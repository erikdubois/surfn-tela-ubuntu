A colour variant of the [Surfn](https://github.com/erikdubois/surfn) icon theme. Ships only the **Surfn-Tela-Ubuntu** folder (places) icons and inherits the base **Surfn** icon set, so `surfn-icons-git` is required. Folder icons are sourced from [vinceliuice/Tela-icon-theme](https://github.com/vinceliuice/Tela-icon-theme).

## Installation (Arch / Kiro — nemesis_repo)

```
sudo pacman -S surfn-tela-ubuntu-icons-git
```
